# 🍽️ Zomato Data Analysis Project

## 📌 Project Overview

This project analyzes food delivery data to understand customer behavior, restaurant performance, and city-wise demand patterns. The goal is to extract meaningful insights that improve delivery efficiency and customer satisfaction.

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy, Matplotlib)
* SQL
* Power BI
* Google Colab

---

## 📊 Key Analysis Performed

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Top Restaurants based on Ratings
* Most Popular Cuisines
* City-wise Order Distribution
* Cost vs Rating Analysis
* Peak Order Time Analysis

---

## 📈 Key Insights

* High-rated restaurants attract more customers
* Popular cuisines dominate order volume
* Metro cities generate the highest demand
* Peak order time is during evening hours
* Higher cost does not always mean higher rating

---

## 💡 Business Recommendations

* Increase delivery agents during peak hours
* Provide offers during non-peak hours
* Promote high-rated restaurants
* Improve delivery time optimization

---

## 📊 Dashboard

Interactive dashboard created using Power BI showing:

* KPIs (Total Orders, Avg Rating, Delivery Time)
* Filters (City, Cuisine, Rating)

---

## 🚀 Project Outcome

This project demonstrates how data analysis can help optimize food delivery operations and improve customer experience.
